# Deep Learn — 三板斧深度学习

用三个递进式提问快速掌握任意领域。方法论详见 `references/method-origin.md`。

## 用户输入的五种方式

用户可能以任何方式提供学习素材，按以下方式处理：

### 1. 给链接
```bash
python3 scripts/smart_fetch.py "<url>"              # 单个
python3 scripts/smart_fetch.py "<url1>" "<url2>"    # 批量
python3 scripts/smart_fetch.py "<url>" -o output.md # 存文件
```
自动识别平台（微信/B站/YouTube/arXiv/小红书/X/网页），自动选择抓取策略，失败自动 fallback。首次运行自动安装依赖。

### 2. 直接发文字
用户直接在对话中粘贴文章内容、笔记、讲义。无需抓取，直接进入三板斧流程。

### 3. 发文件
- **PDF**：用 `pdf` 工具分析，`pdf(path='用户发送的文件路径', prompt='提取全部内容')`
- **Markdown/TXT**：用 `Read` 工具读取
- **Word/PPT**：提示用户转为 PDF 或粘贴文字

### 4. 发图片（截图/笔记/PPT）
用 `image` 工具提取文字内容：`image(path='图片路径', prompt='提取图中所有文字内容')`

### 5. 只给主题（没有素材）
用户说"帮我学习 XXX"但没有提供任何资料。Agent 应：
1. 用 `web_search` 搜索该主题的权威资料（教程、论文、官方文档）
2. 用 `smart_fetch.py` 抓取搜索到的 top 3-5 个链接
3. 结合 Agent 自身知识，进入三板斧流程

搜索建议 query 模板：
- `"{主题}" 入门教程 最佳资源`
- `"{主题}" comprehensive guide OR tutorial`
- `"{主题}" 论文 survey review`

## 抓取坑点（Agent 必读）

1. **B站视频无法提取字幕**。脚本只能拿标题和简介。若是搬运视频，脚本会自动从简介中检测 YouTube 原版链接并提取字幕。未检测到时，Agent 需手动搜索 YouTube 原版，用 `summarize '<youtube_url>' --youtube auto --length xxl --extract-only` 提取。
2. **微信公众号**：脚本内置三级 fallback，绝大多数能抓到。全失败时请用户复制原文。
3. **arXiv 论文**：脚本拿摘要，建议 Agent 额外用 `pdf` 工具分析全文。
4. **一批资料 3-5 个来源**，总量 5 万字以内。

## 三板斧流程

所有素材就绪后，一次性吃进上下文，依次执行：

### Prompt 1 — 心智模型（骨架）
提炼 5 个核心心智模型。不是知识点列表，是专家看问题的思维结构。每个用隐喻命名，配 3-5 句解释。

### Prompt 2 — 专家分歧（深水区）
找出 3 个根本性分歧。说明各方立场、论据，以及为什么这个分歧重要。

### Prompt 3 — 深度自测（真懂 vs 假懂）
生成 10 道区分深度理解和死记硬背的题目。需要跨概念、类比、反事实思考。附参考答案方向。

## 归档结构

```
~/workspace/learning/{主题名}/
├── README.md                    ← 学习计划总览
├── batch-{N}-{批次名}/
│   ├── report.md                ← 三板斧报告
│   └── sources/
│       ├── README.md            ← 资料链接索引
│       └── {来源}-{类型}.md     ← 文字稿/摘要
```

```markdown
# 第 N 批学习报告 — {批次名}

> 完成时间：{日期}
> 资料：{列表}

## 🧠 Prompt 1：五大核心心智模型
### 1. 「{隐喻名}」模型
{解释}

## 🔥 Prompt 2：顶尖专家的根本分歧
### 分歧 1：{主题}
- **{立场A}**：...
- **{立场B}**：...

## 📝 Prompt 3：10 道深度自测题
**1.** {问题}

## 参考答案提示
**Q1 方向**：{2-3句}
```

## 互动扩展

用户可以：
- 回答自测题 → Agent 批改打分，指出盲区
- 展开讲解某个心智模型
- 生成更多练习题
- 上传报告到飞书文档
